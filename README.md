# Barbell Factor Strategy: Dividend + Small-Cap
## 红利-小市值杠铃因子策略

> 本项目构建「防御+进攻」双端杠铃策略，以高股息红利资产为盾、小市值资产为矛，通过风格互补实现跨市场环境的风险收益平衡。

---

### 1. 背景与动机（Motivation）
单一风格策略存在明显短板：
- 纯红利策略在成长行情中缺乏收益弹性
- 纯小市值策略在下行周期回撤风险过高
本策略旨在填补单一风格的局限性，打造适配多市场环境的攻守兼备组合。

---

### 2. 策略设计（Methodology）
采用杠铃式配置，构建双端互补结构：
- **防御端（红利因子）**：配置高股息、低波动、现金流稳定标的，提供下行保护与基础收益
- **进攻端（小市值因子）**：配置低市值、具备反转/成长弹性标的，捕捉风格切换超额收益
- 定期再平衡：维持两端目标暴露，实现轮动中的「高抛低吸」收益

---

### 3. 核心发现（Key Insights）
红利因子的效果天然较好；
纯粹小市值因子在某段时间有效但是效果及其微弱，应该配合动量因子组合；
由于缺乏ST股票相关数据，本策略没法剔除ST股票，后续需要继续从外界数据源获取ST相关数据，以完善策略效果。
